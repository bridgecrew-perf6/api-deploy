Currencies List

This returns the list of available currencies

Endpoint:

GET: /currency/list?format={format}

Parameters:

  • Format: JSON or XML

Example Response:

{
    "status":"success",
    "currencies":[
        "AUD":"Australian Dollar",
        "BGN":"Bulgarian Lev",
        [ ... ]
    ]
}
